# GeoShot — Plan
## Conceito do Jogo

GeoShot é um top-down shooter 2D para iOS em que o jogador controla uma seta dentro de uma dungeon com 3 andares. Os inimigos são formas geométricas, cada uma com comportamentos e estatísticas diferentes. Não existem armas — as balas saem diretamente da seta na direção em que o jogador está a apontar.

O estilo visual é minimalista e intencional: tudo são formas geométricas puras, sem texturas nem sprites. O jogo é construído em Swift com SpriteKit.

## Como se Joga

O jogador usa dois joysticks virtuais:

  • Joystick esquerdo — mover o jogador livremente pelo ecrã.
  • Joystick direito — apontar a direção. As balas saem automaticamente enquanto o joystick estiver ativo.

O mapa é uma dungeon com 3 andares. Cada andar tem 4–5 salas de combate ligadas por corredores e um boss obrigatório no fim. Quando o jogador entra numa sala, as portas fecham. Só abrem após derrotar todos os inimigos. No fim de cada sala, o jogador recebe uma recompensa aleatória.

## As Formas
### Jogador
  • Arrow (Seta) — Estatísticas equilibradas. Dispara na direção do joystick direito. Efeito bullet web entre projéteis.
### Inimigos
  • Triangle — Muito rápido, pouca vida. Persegue e tenta fazer contacto. Cor: laranja.
  • Plus (+) — Médio. Dispara 4 balas em simultâneo para os 4 lados. Cor: vermelho.
  • Square — Muito lento, muita vida. Uma bala pesada em direção ao jogador. Cor: branco.
  • Circle — Mantém distância. Bala rápida de longo alcance. Cor: roxo.
### Bosses por Andar
  • Andar 1 — Giant Square (HP ×3, dano ×2)
  • Andar 2 — Mega Plus (dispara para 8 direções)
  • Andar 3 — Hexagon (boss final — combina todos os ataques, tem fases)
## Upgrades
### Base (por sala)
  • +Vida — aumenta a vida máxima
  • +Balas — mais projéteis por tiro
  • +Dano — cada bala causa mais dano
  • +Velocidade — move-se mais depressa
  • +Velocidade de Disparo — cadência maior
### Especiais
  • Escudo — anel geométrico que absorve hits. Cooldown para regenerar.
  • Power-Up Temporário — botão no HUD. Tipos: Overdrive, Ghost, Nova, Time Slow, Magnet.
  • Ricochete — balas saltam nas paredes uma vez.
  • Perfurante — balas atravessam múltiplos inimigos.
  • Regeneração — +1 HP a cada 8s sem receber dano.
  • Decoy — seta falsa que distrai inimigos por 6s.
## Gameplay Loop

Explorar → Entrar sala → Portas fecham → Matar todos → Upgrade → Avançar → Boss de andar → Próximo andar → Boss final

## Design Visual
  • Fundo escuro com grid subtil de pontos
  • Personagem: triângulo cyan com efeito bullet web entre balas
  • Inimigos: cada cor é única e imediatamente reconhecível
  • HUD: corações de vida, timer, contador de sala, slots de upgrade
  • Explosões: burst de partículas na cor do inimigo destruído
  • Screen shake ao receber dano
## MVP
  • Arrow player com dual joystick + disparo automático
  • 2 tipos de inimigos: Triangle + Plus
  • 5 salas lineares + 1 boss (Giant Square)
  • 5 upgrades base funcionais
  • HUD simples com vida + indicador de sala
  • Game Over + Victory screen
